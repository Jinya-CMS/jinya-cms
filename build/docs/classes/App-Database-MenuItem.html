public string|null $route = null

The link of the menu item. This link can be either an absolute URL, a relative URL or null. Absolute URLs open the link directly, while relative URLs are handled by the frontend controller and will render different entities. Usually a null value indicates that the menu item is used as a parent item, which has no link but rather opens and shows the sub items
